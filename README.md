# Food Store – Sistema de Gestión de Pedidos

## Descripción
Sistema de consola en Java para gestionar categorías, productos, usuarios y pedidos. 
Desarrollado como TPI de Programación 2 – UTN.

## Requisitos
- Java 21
- NetBeans IDE (o cualquier IDE compatible con Java)

## Cómo ejecutar
1. Clonar el repositorio
2. Abrir el proyecto en NetBeans
3. Ejecutar el archivo `Integrador_programacion_2.java` con F6

## Tecnologías
- Java 21
- Colecciones (ArrayList)
- POO: herencia, interfaces, excepciones propias
